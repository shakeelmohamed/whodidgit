Who Did Git?



Sometimes commits show up with an unknown author, like above.
This tool helps you investigate the the problem!

Try it with this form!




Do an HTTP request on this domain with the following parameters:

  • owner (the GitHub repository owner or organization)
  • repo (the name of a public GitHub repository)
  • sha (the sha of the individual GitHub commit)
  • token (optional. a GitHub access token for private repos or increased API limits)
