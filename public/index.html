Who Did Git?

Sometimes commits show up with an unknown author, like so:



This tool helps you investigate the the problem!

Try it with this form!




Do an HTTP request on this domain with the following parameters:

  • owner (the GitHub repository owner or organization)
  • repo (the name of a public GitHub repository)
  • sha (the sha of the individual GitHub commit

And we'll give you some JSON from the GitHub API about the author of that commit.


Here's an example:

?owner=shakeelmohamed&repo=whodidgit&sha=c977b473efb8d1fa3bede0171b005bb4c00c1561

Which would return:

GitHub gave us this information about the author of that commit: 

{"name":"unknown","email":"Shakeel@Example.com","date":"2015-01-29T08:32:42Z"}
